### 知识点解析  
（注：基于上文消费者效用最大化核心框架，本测验聚焦参数调整与函数形式变化的实操应用，无图表故省略图形解释部分）  

#### 5.1 题解析  
- **理论衔接**：延续上文效用最大化条件 $\text{MRS} = \frac{p_A}{p_B}$ 与预算约束 $p_A x_A + p_B x_B = m$。  
- **关键逻辑**：  
  - 给定 $U = x_A x_B$，可得 $\text{MRS} = \frac{x_B}{x_A}$。  
  - 代入价格比 $\frac{p_A}{p_B} = \frac{1}{2}$，导出最优比例 $x_A = 2x_B$。  
  - 结合预算约束 $x_A + 2x_B = 240$，解出香蕉消费量。  
- **设计意图**：强调Cobb-Douglas效用函数中收入分配比例的稳定性（收入半数用于香蕉），呼应上文“最优消费束与价格-收入参数的线性关系”结论。  

#### 5.2 题解析  
- **理论衔接**：在5.1效用函数基础上修改价格与收入，检验预算约束变动下的需求重分配。  
- **关键逻辑**：  
  - 新预算约束 $5x_A + 6x_B = 40$ 与 $\text{MRS} = \frac{x_B}{x_A} = \frac{5}{6}$ 联立。  
  - 求解苹果消费量需验证非负约束（因收入较低，需排除无解风险）。  
- **设计意图**：突出价格上升对需求抑制的非线性效应（原苹果价格1→新价格5），深化“价格变动对需求的边际影响随参数变化”的讨论。  

#### 5.3 题解析  
- **理论衔接**：转换效用函数形式 $U = (X+2)(Y+1)$，需重新推导 $\text{MRS}$。  
- **关键逻辑**：  
  - 计算 $\text{MRS} = -\frac{MU_X}{MU_Y} = -\frac{Y+1}{X+2}$，结合题设 $\text{MRS} = -2$ 与 $X=10$。  
  - 直接解方程 $-(Y+1)/(10+2) = -2$，避免预算约束介入（题干仅关注偏好数学性质）。  
- **设计意图**：通过平移效用函数检验对 $\text{MRS}$ 本质的理解——平移不改变边际替代率的比值特性，但改变绝对量计算，强化上文“效用函数单调变换不影响MRS”的结论。

### 题目关联性  
三题构成**参数-形式双维度演进链**：  
1. 5.1 以基础Cobb-Douglas函数建立效用最大化标准流程；  
2. 5.2 通过价格/收入扰动检验上文“预算线移动对最优解的影响”；  
3. 5.3 用非标准效用函数验证上文“MRS独立于效用函数线性变换”的核心定理。  
整体体现微观经济学中“约束条件变化→需求响应→偏好数学表征”的逻辑递进。

### 题目解析（衔接上文知识点）

#### 5.4 题解析  
- **理论衔接**：延续上文效用最大化条件 $|\text{MRS}| = p_1 / p_2$，但效用函数形式为**拟线性**（quasilinear），其中 $x_2$ 的边际效用恒为 1。这与上文 5.3 题的平移效用函数 $U = (X+2)(Y+1)$ 形成对比——拟线性函数导致一种商品（$x_2$）的需求**完全不依赖收入**，仅由相对价格决定。  
- **关键逻辑**：  
  - 计算 $\text{MRS} = -\frac{MU_{x_1}}{MU_{x_2}} = -\frac{2 / \sqrt{x_1}}{1} = -2 / \sqrt{x_1}$。  
  - 代入 $p_1 / p_2 = 1/4$，得最优条件 $2 / \sqrt{x_1} = 1/4$，解得 $x_1 = 64$（坚果消费量与收入无关，印证拟线性需求特性）。  
  - 验证预算约束 $x_1 + 4x_2 = 72$，但问题仅问 $x_1$，故无需计算 $x_2$。  
- **与上文演进关系**：作为 5.3 题的延伸，本题通过非 Cobb-Douglas 函数验证 **MRS 仅取决于边际效用比值** 的普适性，呼应上文“偏好数学表征决定需求结构”的核心定理。

#### 5.5 题解析  
- **理论衔接**：同一效用函数下，收入变动（72→100）仅影响 $x_2$ 消费量，符合拟线性效用的**收入效应分离特性**（上文未覆盖，但隐含于 5.2 题价格变动分析框架）。  
- **关键逻辑**：  
  - 由 5.4 题最优条件 $x_1 = 64$ 恒成立（与收入无关），代入新预算约束 $64 + 4x_2 = 100$，解得 $x_2 = 9$。  
  - 此处 $x_2$ 随收入线性增长，凸显拟线性函数中“收入仅影响非线性商品外的消费”规律，深化上文“预算约束变动导致需求重分配”结论。  
- **设计意图**：通过固定 $x_1$ 需求，检验对拟线性效用中 **收入效应归零** 的理解，补全 5.2 题价格变动外的收入维度分析。

#### 5.6 题解析  
- **理论衔接**：引入 **Leontief 型效用函数 $U = \min\{x, y^2\}$**，需放弃 MRS 条件，转用固定比例约束 $x = y^2$。这与上文 5.1-5.3 题的凸偏好形成分水岭，验证“**非凸偏好需特殊处理**”的扩展定理（上文隐含：偏好类型决定求解方法）。  
- **关键逻辑**：  
  - 由 $y = 7$ 代入固定比例条件得 $x = y^2 = 49$。  
  - 收入 $m = p_x x + p_y y = 15 \times 49 + 10 \times 7 = 805$，无需边际分析。  
- **与上文演进关系**：作为参数-形式双维链的收尾，本题通过 **$\min\{x, y^2\}$ 形式** 展示上文“效用函数形态直接决定需求性质”结论——固定比例偏好下，收入与价格仅影响消费束规模，不影响比例，呼应 5.1-5.2 题中 Cobb-Douglas 的比例稳定性，但机制迥异。
