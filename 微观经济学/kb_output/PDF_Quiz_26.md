# Factor Markets 问题解析

## 1. 文字与公式提取

### 26.1 题目信息
- **需求曲线**：$ p = 30 - 12q $（$ p $ 为消费者支付单价，$ q $ 为购买量）
- **市场结构**：上游垄断生产商（零成本）向下游垄断分销商批发，分销商向消费者销售
- **生产商决策**：设定批发价 $ c $ 以最大化自身总收入
- **分销商决策**：以 $ c $ 为边际成本选择产出
- **选项**：  
  (a) 15.  
  (b) 22.50.  
  (c) 2.50.  
  (d) 1.25.  
  (e) 7.50.

### 26.2 题目信息
- **劳动供给曲线**：$ w = 60 + 0.08L $（$ L $ 为周劳动单位，$ w $ 为周工资）
- **当前雇佣量**：$ L = 1,000 $ 单位/周
- **选项**：  
  (a) equals the wage rate.  
  (b) is twice the wage rate.  
  (c) equals the wage rate plus 160.  
  (d) equals the wage rate plus 80.  
  (e) equals the wage rate plus 240.

### 26.3 题目信息
- **生产函数**：$ Q = 40L $（$ L $ 为劳动投入，$ Q $ 为产量）
- **劳动供给**：$ w = 40 + 0.1L $（雇佣 $ L $ 单位劳动需支付的工资）
- **产品需求曲线**：$ P = 30.75 - \frac{Q}{1,000} $
- **问题导向**：求利润最大化产量

## 2. 图表分析与经济逻辑

> **当前图片无图表**，重点解析三道题目中的**要素市场典型模型**：

### 26.1 模型：双向垄断（Upstream-Downstream Monopoly）
- **核心逻辑**：  
  生产商的定价决策受分销商需求约束。分销商作为垄断中间商，其边际收益 $ MR_d = 30 - 24q $ 即为生产商面临的需求曲线。
- **均衡推导**：
  1. 分销商利润最大化：$ MR_d = c \implies c = 30 - 24q $
  2. 生产商最大化收入（零成本）：$ R = cq = (30 - 24q)q $
     - 一阶条件：$ 30 - 48q = 0 \implies q = \frac{5}{8} $
     - 最优批发价 $ c = 30 - 24 \times \frac{5}{8} = 15 $
  3. 消费者价格：$ p = 30 - 12 \times \frac{5}{8} = 22.50 $
- **结论**：垄断链的双重加成导致消费者价格（22.50）显著高于单层垄断价格，验证 **(b) 22.50** 为正确选项。

### 26.2 模型：买方垄断的边际要素成本（Monopsony MFC）
- **核心逻辑**：  
  单一买家（大学）面对向上倾斜的供给曲线时，雇佣额外劳动的边际成本高于当前工资率。
- **公式推导**：
  - 总成本 $ TC = wL = (60 + 0.08L)L = 60L + 0.08L^2 $
  - 边际要素成本 $ MFC = \frac{dTC}{dL} = 60 + 0.16L $
  - 代入 $ L = 1,000 $：$ w = 60 + 80 = 140 $，$ MFC = 60 + 160 = 220 $
  - 差值：$ MFC - w = 80 $
- **结论**：买方垄断企业需为新增劳动支付原工资 + 额外补偿（此处为80），故 **(d)** 正确。

### 26.3 模型：垄断厂商劳动投入优化
- **核心逻辑**：  
  需通过**生产函数**将劳动投入转化为产出，再结合**劳动供给成本**与**产品需求**求解利润最大化点。
- **联立方程**：
  | **环节** | **公式转换** | **关键表达式** |
  |---|---|---|
  | 劳动-产出 | $ L = \frac{Q}{40} $ |  |
  | 劳动成本 | $ w = 40 + 0.1 \times \frac{Q}{40} = 40 + \frac{Q}{400} $ |  |
  | 总成本 | $ TC = wL = \left(40 + \frac{Q}{400}\right)\frac{Q}{40} $ | $ TC = Q + \frac{Q^2}{16,000} $ |
  | 边际成本 | $ MC = \frac{dTC}{dQ} $ | $ MC = 1 + \frac{Q}{8,000} $ |
  | 边际收益 | $ MR = \frac{d}{dQ}\left(30.75Q - \frac{Q^2}{1,000}\right) $ | $ MR = 30.75 - \frac{Q}{500} $ |
- **均衡条件**：  
  $ MR = MC \implies 30.75 - \frac{Q}{500} = 1 + \frac{Q}{8,000} $  
  解得 $ Q = 14,000 $
- **逻辑延伸**：垄断企业同时面临产品端降价约束与要素端成本递增，需通过$ MR = MC $校准最优产出。

---

## 3. 方法论总结
- **双向垄断（26.1）**：上游厂商的决策基于下游厂商的**边际收益曲线**，而非最终消费者需求
- **买方垄断（26.2）**：$ MFC > w $ 是单雇主劳动力市场的核心特征，差额大小由供给弹性的倒数决定
- **要素-产品联动（26.3）**：垄断厂商利润最大化需**同时满足**产品市场 $ MR=MC $ 与要素市场 $ MRP = MFC $（本题中通过生产函数隐含转化）

> 所有题目聚焦**要素市场中的市场势力传导**：26.1 体现纵向结构下的双重加成，26.2 展示单买方市场的要素定价扭曲，26.3 验证垄断企业通过生产函数将要素成本转化为产品决策的完整链条。

### 解析与经济逻辑  
- **背景关联**：  
  本图片为**问题26.3的备选项**，源自上文“垄断厂商劳动投入优化”模型（见26.3部分）。根据上文推导，垄断厂商通过联立生产函数与要素成本曲线，解得利润最大化产量 \( Q = 14,000 \)（对应选项 (a)）。  

- **选项验证**：  
  - 上文已证明：均衡条件 \( MR = MC \) 导出 \( Q = 14,000 \)，其中：  
    - 产品需求曲线 \( P = 30.75 - \frac{Q}{1,000} \) 决定了边际收益 \( MR = 30.75 - \frac{Q}{500} \)  
    - 劳动供给约束（\( w = 40 + 0.1L \)）与生产函数 \( Q = 40L \) 推导出边际成本 \( MC = 1 + \frac{Q}{8,000} \)  
  - 其他选项 (b)-(e) 均源于关键参数误设：  
    - (b) 28,000：若忽略劳动供给成本递增（即假设 \( MC \) 恒定），但题目中 \( w \) 随 \( L \) 上升，故排除。  
    - (c)-(e)：数值过小，可能源自错误代入生产函数（如误用 \( L = Q \) 代替 \( L = \frac{Q}{40} \)）或需求曲线线性化偏差。  

- **结论**：  
  选项 (a) 14,000 是唯一符合**要素-产品市场联动均衡**的解，体现垄断厂商在双重约束（产品端需求递减、要素端供给递增）下对 \( MR = MC \) 的精确校准，与上文26.3模型结论完全一致。
