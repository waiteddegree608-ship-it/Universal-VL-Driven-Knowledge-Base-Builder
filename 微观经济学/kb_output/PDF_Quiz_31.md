##### 问题 31.1：Borda 计数的应用  
- **核心原理**：Borda 计分法总分越低者胜（第一选择权重最低）。  
- **关键计算**（基于上文投票规则）：  
  - **x 总分**：  
    $ (7 \times 1) + (4 \times 1) + (6 \times 3) + (8 \times 3) = 7 + 4 + 18 + 24 = 53 $  
  - **y 总分**：  
    $ (7 \times 2) + (4 \times 3) + (6 \times 2) + (8 \times 1) = 14 + 12 + 12 + 8 = 46 $  
  - **z 总分**：  
    $ (7 \times 3) + (4 \times 2) + (6 \times 1) + (8 \times 2) = 21 + 8 + 6 + 16 = 51 $  
- **结论**：y 以最低分 46 胜出（选项 (b)）。  
- **延伸**：反映社会选择规则中“层级偏好聚合”特性，与上文 Arrow 不可能性定理逻辑关联，但因计分法允许部分排序，避免了完全排序下的循环矛盾。  

##### 问题 31.2：跨区域效用最大化  
- **约束转化**（结合生活成本差异）：  
  - 实际消费量与货币关系：  
    $ C_A = \frac{m_A + 3,000}{p_A}, \quad C_B = \frac{m_B + 9,000}{p_B} = \frac{m_B + 9,000}{3 p_A} $  
    简化为 $ C_A \propto m_A $, $ C_B \propto \frac{m_B}{3} $（设 $ p_A = 1 $）。  
  - 效用函数重写：  
    $$
    U = C_A C_B = \frac{(3,000 + m_A)(9,000 + m_B)}{3}
    $$  
    约束 $ m_A + m_B = 4,000 $。  
- **最优条件**（基于上文效用最大化）：  
  - 边际效用均等化（考虑价格调整）：  
    $$
    \frac{\partial U / \partial m_A}{\partial U / \partial m_B} = \frac{p_B}{p_A} = 3 \implies \frac{C_B}{C_A} = 3 \cdot \frac{C_A}{C_B} \quad \text{（推导得 } m_A = m_B\text{）}
    $$  
  - **经济含义**：尽管 B 城生活成本高，等量货币使 $ C_B $ 低于 $ C_A $，但权重调整后货币分配应均等。选项 (a) 正确。  
  - **策略本质**：货币分配需补偿成本差异，但效用函数对称性使均等分配占优。  

##### 问题 31.3：线性效用下的公平分配  
- **偏好特征**：  
  - Paul 的 MRS：$ \frac{MU_{A_P}}{MU_{O_P}} = \frac{5}{1} = 5 $ → 强烈偏好苹果。  
  - David 的 MRS：$ \frac{MU_{A_D}}{MU_{O_D}} = \frac{1}{5} = 0.2 $ → 强烈偏好橙子。  
- **帕累托前沿**（基于上文公平分配框架）：  
  - 线性效用无内部切点，帕累托最优点位于边界：  
    - Paul 消费全部 8 苹果，David 消费全部 8 橙子。  
    - 或按比例分配使双方效用满足特定约束（如 $ 5A_P + O_P = A_D + 5O_D $）。  
- **“fair” 潜在条件**：  
  - 效用均等化 $ 5A_P + (8 - O_P) = (8 - A_P) + 5O_P $ → 要求 $ A_P + O_P = 8 $。  
  - 与上文“公平分配需同时满足帕累托最优与个体理性”一致，但需结合资源总量求解。

### 2. 结合上文知识点的解析
#### 问题 31.3 相关选项（公平分配条件）
上文知识点总结（问题 31.3）明确：Fair 分配需满足效用均等化 $ 5A_P + O_P = A_D + 5O_D $，结合总资源约束 $ A_P + A_D = 8 $ 与 $ O_P + O_D = 8 $，推导出 $ A_P + O_P = 8 $。此条件确保 Paul 与 David 效用相等，且符合帕累托最优边界（线性效用无内部切点，需在边界满足公平性）。

- **(a) $ A_D = A_P $ 且 $ O_D = O_P $**：  
  此为消费量对称，但 **不代表公平分配**。上文指出，Fair 条件仅需效用均等，消费量可不对称（如 Paul 全苹果、David 全橙子仍满足 $ A_P + O_P = 8 $）。若 $ A_P = A_D = 4 $ 且 $ O_P = O_D = 4 $，效用相等（Paul: $ 5 \times 4 + 4 = 24 $，David: $ 4 + 5 \times 4 = 24 $），但非唯一解。**不符合上文核心逻辑**：公平性不依赖消费量均等，而依赖效用均等。

- **(b) $ 10A_P + 2O_P \geq 48 $ 且 $ 2A_D + 10O_D \geq 48 $**：  
  由效用函数，$ 10A_P + 2O_P = 2(5A_P + O_P) = 2U_P $，$ 2A_D + 10O_D = 2(A_D + 5O_D) = 2U_D $。问题要求 $ U_P, U_D \geq 24 $，但效用范围仅 $ [8, 40] $（因总资源限制），**48 超出最大值 40，不成立**。上文明确 Fair 条件仅需 $ U_P = U_D $，无最低效用约束。

- **(c) $ 5A_P + O_P \geq 48 $ 且 $ 2A_D + 5O_D \geq 48 $**：  
  $ 5A_P + O_P = U_P \leq 40 $，无法 $\geq 48$；$ 2A_D + 5O_D $ 与 David 效用 $ U_D = A_D + 5O_D $ 无直接关系，**违反资源约束与效用定义**。上文推导中效用均等化是核心，无此数值下限。

- **(d) $ A_D + O_D \geq 8 $ 且 $ A_S + O_S \geq 8 $**：  
  $ A_D + O_D \leq 8 $（总资源 $ A_P + A_D + O_P + O_D = 16 $，但个体消费和无上限约束），且 $ A_S + O_S $ 为笔误（上文无 $ S $ 变量）。上文 Fair 条件基于效用均等，**非消费总量约束**，且总资源固定，此选项无经济意义。

- **(e) $ 5A_P + O_P \geq A_D + 5O_D $ 且 $ A_D + 5O_D \geq 5A_P + O_P $**：  
  **直接等价于效用均等化 $ U_P = U_D $**，与上文知识点总结完全一致。结合资源约束，此条件推出 $ A_P + O_P = 8 $，涵盖所有帕累托最优且公平的分配（如 $ A_P = 8, O_P = 0 $ 或 $ A_P = 0, O_P = 8 $ 均满足）。**正确体现上文 Fair 分配的核心逻辑**：效用相等是公平分配的必要条件。

#### 问题 31.4（外部性下的自愿交换）
上文知识点总结强调社会选择中个体偏好聚合（如 Borda 计分法）及效用最大化需考虑边际条件。本题中，效用函数含对方消费（$ U_R $ 依赖 $ S_J $，$ U_J $ 依赖 $ S_R $），体现正外部性（双方偏好对方消费增加）。

- **关键机制**：个体愿转移资源当其边际收益大于边际成本。  
  - Romeo 转移条件：$ \frac{\partial U_R}{\partial S_J} > \frac{\partial U_R}{\partial S_R} \implies 4 S_R^8 S_J^3 > 8 S_R^7 S_J^4 \implies S_R > 2 S_J $。  
    代入 $ S_J = 96 - S_R $，得 $ S_R > 64 $。
  - Juliet 转移条件：$ \frac{\partial U_J}{\partial S_R} > \frac{\partial U_J}{\partial S_J} \implies 4 S_R^3 S_J^8 > 8 S_R^4 S_J^7 \implies S_J > 2 S_R \implies S_J > 64 $。

- **选项解析**：  
  - (a)、(d) 错误：Romeo 愿转移需 $ S_R > 64 $，非 48 或 60。  
  - (b) 错误：Juliet 愿转移需 $ S_J > 64 $，非 62。  
  - (c) 错误：当 $ S_R \in [32, 64] $ 时，双方均不愿转移（边界处效用梯度为零），存在分歧可能。  
  - **(e) 正确**：Juliet 有 $ S_J > 64 $ 时，$ S_R = 96 - S_J < 32 $，满足 $ S_J > 2 S_R $，愿转移给 Romeo。**验证上文边际条件逻辑**：效用最大化依赖个体边际替代率，符合条件的分配可避免社会选择循环矛盾。

#### 问题 31.5（带外部性的公平分配）
上文知识点总结（问题 31.3）指出，线性效用下公平分配需效用均等化。本题效用函数含外部性（$ U_H $ 依赖 $ W_M $，$ U_M $ 依赖 $ W_H $），且 sheriff 要求 "both as happy as possible" 与 "treat them equally"，即 **最大化 $ \min(U_H, U_M) $ 且 $ U_H = U_M $**。

- **公平条件**：  
  设 $ U_H = U_M \implies W_H - W_M^{1/4} = W_M - W_H^{1/4} $。  
  令 $ X = W_H^{1/4}, Y = W_M^{1/4} $，化简得 $ (X - Y)[(X+Y)(X^2+Y^2) + 1] = 0 \implies X = Y \implies W_H = W_M $（因括号项恒正）。

- **最优化**：  
  令 $ W_H = W_M = c $，则 $ U_H = U_M = c - c^{1/4} $。  
  边际效用 $ \frac{dU}{dc} = 1 - \frac{1}{4} c^{-3/4} > 0 $ 当 $ c > \left(\frac{1}{4}\right)^{4/3} \approx 0.157 $，故 $ U $ 随 $ c $ 增加而增加。  
  为最大化效用，需 $ c $ 尽可能大，即 **$ W_H + W_M = 28 $（无倾倒）且 $ W_H = W_M = 14 $**。  
  此时 $ U_H = U_M = 14 - 14^{1/4} \approx 12.07 $；若倾倒（如其他选项），$ c < 14 $，效用降低。

- **选项解析**：  
  - **(a) 正确**：无倾倒且消费均等，符合效用均等化与效用最大化（上文 31.3 的公平分配逻辑延伸）：个体效用相等且总效用最大。  
  - (b)–(e) 错误：倾倒降低总消费量，减少 $ c $，违反 "as happy as possible"。上文已强调，公平分配需满足帕累托最优与个体理性，倾倒导致无谓损失。  
  **关键差异**：不同于问题 31.3 的线性效用（无外部性），本题外部性使公平条件仍导向消费均等（因效用函数对称），延续上文社会选择中 "效用对称性" 的隐含假设。
